Data Flow Example

1
Photon2 publishes: bootcamp/alice/photon2/temperature → "72.5"
2
Mosquitto routes message to all subscribers
3
Backend receives message via MQTT subscription
4
Backend saves to TimescaleDB: INSERT INTO sensor_data ...
5
Backend pushes update to dashboard via WebSocket
6
Dashboard updates chart in real-time

MQTT Topic Structure

Pattern:
bootcamp/{'{username}'}/{'{device}'}/{'{sensor}'}

ACL Rules:
