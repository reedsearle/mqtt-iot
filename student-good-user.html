Student Good User Persona

Name

Sally Student

Technology

Devices

Windows or

Mac

Love/Hate

Loves to learn

Wants to learn IoT

Proficiency

Understands how to use her computer to access the internet.

Might have taken a class in another programming language.

Attitudes & Behaviours

User Story

As a student, I want to learn how to build and program an IoT device.

Use Case

Sally has an assignment to build an IoT device and connect it to the MQTT broker to see her data online.

Preconditions

Sally has an IoT device built according to the lesson plan to transmit data to the MQTT broker.

Sally has her browser open to the MQTT broker login page.

Post Conditions

Sally has beginner's experience in connecting her IoT device to the MQTT broker.

Interaction Flow

Sally creates a login profile on the MQTT broker.

Sally logs into the MQTT broker.

Sally creates a test project.

Sally adds feeds to her test project.

Sally creates a dashboard for her test project.

Sally sends data to her test project.
